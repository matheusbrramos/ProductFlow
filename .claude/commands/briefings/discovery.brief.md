# /discovery - Briefing

**Foco**: Pesquisa qualitativa de usuários
**Fase**: 2 (após /researcher)
**Depende**: `.context/empresa.md`, contexto de /researcher

## Comandos
- `/analyze <dados>` - Analisa transcrições/surveys
- `/interview <perfil>` - Prepara guia de entrevista
- `/ost <outcome>` - Cria Opportunity Solution Tree
- `/jtbd <segmento>` - Mapeia Jobs-to-be-Done
- `/synthesize` - Consolida insights

## Output
- `docs/discovery/` - Análises, OST, JTBD, guias

## O que faz
- Processa dados qualitativos automaticamente
- Extrai quotes e identifica padrões
- Mapeia oportunidades (Teresa Torres)
- Conecta usuários a outcomes de negócio

## Não faz
- Pesquisa de mercado → /researcher
- Criar PRD → /strategist

→ Detalhes: `.claude/commands/agents/discovery.md`
