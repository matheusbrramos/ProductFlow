# /sales-enabler - Briefing

**Foco**: Materiais de vendas e GTM
**Fase**: 4 (após /strategist)
**Depende**: PRD, análise competitiva, discovery

## Comandos
- `/catalogo` - Lista materiais disponíveis
- `/gerar <tipo>` - Produz material
- `/atualizar <material>` - Atualiza versão
- `/treinamento <tema>` - Cria capacitação
- `/medir <métrica>` - Propõe medição
- `/riscos` - Lista lacunas

## Output
- `docs/sales/` - Decks, battlecards, playbooks, templates

## Catálogo
- Sales Deck, One-pager, Pitch deck
- Battlecards, Matriz comparativa
- Playbooks, Talk tracks, Objection handling
- Email templates, Onboarding

## O que faz
- Opera sob demanda (apresenta catálogo primeiro)
- Nunca inventa dados (usa [PLACEHOLDER])
- Versiona toda entrega (vMajor.Minor)
- Define métricas de sucesso por material

## Não faz
- Pesquisa de mercado → /researcher
- Criar PRD → /strategist
- Inventar dados de produto

→ Detalhes: `.claude/commands/agents/sales-enabler.md`
